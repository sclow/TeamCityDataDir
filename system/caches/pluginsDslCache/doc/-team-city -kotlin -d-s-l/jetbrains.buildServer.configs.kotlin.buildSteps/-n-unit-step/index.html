A build step running NUnit tests

Example. Runs NUnit tests with help of the default NUnit console tool installed on the TeamCity server. Collects —Åode coverage with help of JetBrains DotCover and reports it to the TeamCity server together with the test results.

nunit {
  nunitPath = "%teamcity.tool.NUnit.Console.DEFAULT%"
  includeTests = """tests\*.dll"""
  coverage = dotcover {
    assemblyFilters = "+:*"
  }
}
